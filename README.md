# Customer Churn Prediction & Analytics

A data analytics and machine learning project to predict customer churn in the telecommunications industry.

## Features
- **Machine Learning Model**: Automatically predicts if a customer is likely to cancel their subscription based on their profile.
- **Interactive Dashboard**: A full web application built with Streamlit featuring live KPI metrics and dynamic charts.
- **Power BI Explorer**: A drag-and-drop data visualization tool integrated directly into the app using PyGWalker.
